Proyecto SQL – Día 7 (NetflixDB)

Objetivo
Identificar las series más exitosas en la base de datos NetflixDB, considerando:
- La cantidad total de episodios.
- El rating promedio en IMDB

Resultado esperado
Una tabla con:
1. Título de la serie  
2. Cantidad de episodios  
3. Rating promedio  

Ordenada de mayor a menor rating.
